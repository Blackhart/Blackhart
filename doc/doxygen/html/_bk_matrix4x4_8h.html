BK_API struct BkMatrix4x4 BkMatrix4x4_Mul_BkMatrix4x4 ( struct BkMatrix4x4 const *  m1,
struct BkMatrix4x4 const *  m2 
)

Multiplies a 4x4 matrix with another 4x4 matrix.

Performs the following operation: ret = m1 * m2.

Matrix multiplication isn't commutative; m1 * m2 =/= m2 * m1.

Matrix multiplication is associative; (m1 * m2) * m3 == m1 * (m2 * m3).

Parameters
m1The first 4x4 matrix.
m2The second 4x4 matrix.
Returns
The multplied 4x4 matrix.
