Returns the difference between two quaternions.

BK_API struct BkQuaternion BkQuaternion_Mul_BkQuaternion ( struct BkQuaternion const *  q1,
struct BkQuaternion const *  q2 
)

Multiplies two quaternions.

Performs the following operation: ret = q1 * q2.

Quaternion multiplication isn't commutative; q1 * q2 =/= q2 * q1.

Quaternion multiplication is associative; (q1 * q2) * q3 == q1 * (q2 * q3).

Parameters
q1The first quaternion to multiply.
q2The second quaternion to multiply.
Returns
The multiplied quaternion.
